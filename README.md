# DATA.ML.360-2023-2024-1


## Docker
Docker is recommended to run this project.

### Build
`docker build -t recommender-system-assignment .`

### Run
`docker run recommender-system-assignment`

### Dev
`docker-compose up`

## Local without Docker

### Install
`npm run install`

### Run
`npm run program`

### Dev
`npm run start`
